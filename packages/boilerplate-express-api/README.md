# Boilerplate: Express API

## Setup

### What's included

Phase 2 boilerplate project with:

 - Express
 - body-parser
 - nodemon
 - SQLite3
 - Knex
 - Jest


### Installation

```
npm install
npm run knex migrate:latest
npm run knex seed:run
npm run dev
```

Be sure to check out the other npm scripts too.
